# SDL_helloworld

This is a simple skeleton program that builds an extremely simple SDL-based
program, setting up CI for GitHub Actions and such.

Feel free to use this as a starting point for your own code.

This is the branch that uses the [SDL's own GPU API](https://wiki.libsdl.org/SDL3/CategoryGPU) as starting point.
There are other branches with starting points for [SDL Renderer](https://github.com/TechnicJelle/SDL_helloworld/tree/main), and [Vulkan](https://github.com/TechnicJelle/SDL_helloworld/tree/vulkan).
